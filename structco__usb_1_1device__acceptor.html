Detailed Description

Wrapper around raw hotplug_awaitable.

This is a convenience wrapper and sarcifices control, namely ability to change events and flags parameters for a better interface matching that of Corosio and Asio.

Note
maybe not needed to be a class, kept as a class for compatibility reasons
